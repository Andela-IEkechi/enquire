Doctopus
======
This README documents whatever steps are necessary to get the application up and running.
#### Ruby version
 * 2.1.2
#### Dependencies
* AWS_ACCESS_SECRET_KEY
* AWS_ACCESS_ID
#### Configuration
 * clone the project
 * run <tt>bundle install</tt>
#### Database creation
 * $ rake db:create
 * $ rake db:migrate
#### How to run the test suite
 * $ rspec
